🦉 *Did you know* CODE LEAP has a simple test for every internal announcement?

It is *the CODE LEAP test*: does this message make someone feel *informed* and *included*? If yes, send it. 💛

*How to use it:* posting to the team? Lead with a clear headline, bold the specifics (dates, rooms, deadlines), and close warm.
